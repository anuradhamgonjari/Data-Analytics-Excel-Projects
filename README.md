1)Overview:

This project is an interactive Production Planning Dashboard built entirely in Microsoft Excel. It helps analyze production performance, inventory levels, sales trends, and regional performance using Pivot Tables, Pivot Charts, Slicers, and advanced Excel features.
The dashboard enables users to monitor production efficiency, compare forecast demand with actual production, identify low-stock items, and analyze product and regional performance.

West region generated the highest sales.
Multiple SKUs are currently categorized as low stock.
Product performance varies significantly across categories.
